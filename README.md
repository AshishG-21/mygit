# Employee Room Assignment System

## Backend (FastAPI)

### Install dependencies
pip install fastapi uvicorn

### Run server
uvicorn main:app --reload

### Open in browser
http://127.0.0.1:8000/docs

---

## Frontend (React)

### Go to frontend folder
cd frontend

### Install dependencies
npm install

### Run app
npm run dev

### Open in browser
http://localhost:5173
