# Employee Room Assignment System

## Backend (FastAPI)

### Install dependencies
Go to command prompt and enter pip install fastapi uvicorn

### Save as main.py
Save the FastAPI code in file main.py

### Run server
Open terminal in VS Studio Code and run -> uvicorn main:app --reload

### Open in browser
http://127.0.0.1:8000/docs

---

## Frontend (React)

### Go to frontend folder
cd frontend

### Install a Build Tool (Vite)
Run this command to install Vite:
npm install -g create-vite

### Create a React Application
Run this command to create a React application named my-react-app:
npm create vite@latest my-react-app -- --template react

### Replace App.jsx
Replace App.jsx code with the new React code

###  Navigate to your new react application directory:
cd my-react-app
cd src

### Install dependencies
npm install

### Run app
npm run dev

### Open in browser
http://localhost:5173
